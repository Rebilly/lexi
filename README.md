### Commands

| command  | description                                                                                               |
| -------- | --------------------------------------------------------------------------------------------------------- |

| `lint`   | Run eslint on all files                                                                                   |
| `format` | Run prettier on all files                                                                                 |
| `build`  | build the dist file. You are required to run this locally in order to build the dist before opening a PR. |
